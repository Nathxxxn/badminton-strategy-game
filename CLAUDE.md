# Badminton Strategy Game

## Objectif
Jeu web éducatif interactif qui enseigne la stratégie du badminton en double.

## Stack technique
- HTML/CSS/JavaScript (single-page app)
- Pas de framework lourd — vanilla JS ou éventuellement React
- Canvas 2D pour le terrain de badminton

## Structure du projet
- index.html — point d'entrée
- /src/js/ — logique du jeu
- /src/css/ — styles
- /assets/ — images, icônes

## Fonctionnalités principales
- Terrain de badminton 2D interactif
- Scénarios de jeu en double (attaque/défense)
- Le joueur choisit où placer le coup
- Feedback immédiat (bon/mauvais choix + explication)
- Progression par niveaux de difficulté

## Conventions
- Code commenté en anglais (projet scolaire en anglais)
- Interface utilisateur en anglais
- Noms de variables descriptifs
